# CyberSentinel UI

A production-ready React TypeScript dashboard for monitoring and analyzing syslog data.

## Quick Start

```bash
npm start
```

Visit `http://localhost:3000`

## Features

- Real-time Dashboard with statistics and charts
- Log Viewer with filtering and search
- Advanced Search with saved queries
- Security Alerts monitoring
- System Settings management
- Dark theme optimized for security operations

## Build

```bash
npm run build
```

## API Configuration

Set in `.env`:
```
REACT_APP_API_URL=http://localhost:8000
```

See full documentation in the application or contact the system administrator.
